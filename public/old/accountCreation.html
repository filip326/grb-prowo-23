Dateien hochladen

Laden Sie bitte eine Datei mit einer Liste der Accounts hoch. Folgendes ist zu beachten:

  • Vorname und Nachname müssen in einer getrennten Spalte sein
  • Alle Accounts müssen in aufeinander folgenden Zeilen auf einer Seite sein
  • Falls Sie eine .csv Datei hochladen, achten Sie darauf, dass ein Semikolon als Trennzeichen verwendet wird
  • Falls Sie eine Excel-Mappe hochladen, wird nur die erste Seite angwandt.
